PostgreSQL to Pervasive SQL Conversion Guide

This guide explains how to convert PostgreSQL queries and functions to Pervasive SQL syntax with examples and workarounds for unsupported functions.

1. Simulating RIGHT

In PostgreSQL, the RIGHT function extracts the last n characters of a string. Pervasive SQL does not support RIGHT, but it can be simulated with SUBSTRING and CHAR_LENGTH:

-- PostgreSQL
RIGHT('foobar', 3); -- Result: "bar"

-- Pervasive SQL Equivalent
SELECT SUBSTRING('foobar', CHAR_LENGTH('foobar') - 3 + 1, 3); -- Result: "bar"
2. Simulating LEFT

In PostgreSQL, the LEFT function extracts the first n characters of a string. In Pervasive SQL, use SUBSTRING with a fixed starting position:

-- PostgreSQL
LEFT('foobar', 3); -- Result: "foo"

-- Pervasive SQL Equivalent
SELECT SUBSTRING('foobar', 1, 3); -- Result: "foo"
3. Simulating RPAD

The RPAD function pads a string to the right with a specified character. Pervasive SQL does not support RPAD, but it can be simulated with SPACE and concatenation:

-- PostgreSQL
RPAD('foo', 5, ' ') -- Result: "foo  "

-- Pervasive SQL Equivalent
SELECT 'foo' || SPACE(5 - CHAR_LENGTH('foo')); -- Result: "foo  "
4. Simulating LPAD

The LPAD function pads a string to the left with a specified character. Pervasive SQL does not support LPAD, but it can be simulated by concatenating spaces and the original string:

-- PostgreSQL
LPAD('foo', 5, ' ') -- Result: "  foo"

-- Pervasive SQL Equivalent
SELECT SPACE(5 - CHAR_LENGTH('foo')) || 'foo'; -- Result: "  foo"
5. Simulating TRIM

Pervasive SQL supports LTRIM and RTRIM but not TRIM. To remove whitespace from both sides of a string, use a combination of LTRIM and RTRIM:

-- PostgreSQL
TRIM('  foo  ') -- Result: "foo"

-- Pervasive SQL Equivalent
SELECT RTRIM(LTRIM('  foo  ')); -- Result: "foo"
6. Simulating Concatenation

Pervasive SQL uses || for concatenation, which is the same as PostgreSQL. Here's an example:

-- PostgreSQL
SELECT 'foo' || 'bar'; -- Result: "foobar"

-- Pervasive SQL Equivalent
SELECT 'foo' || 'bar'; -- Result: "foobar"
